# Aarav Khanal - AI Engineer Portfolio

Welcome to the source code for my personal portfolio website.

This project is a high-performance, cinematic, and interactive portfolio designed to showcase my work in AI Engineering, Machine Learning, and Full-Stack Development. It is built from scratch with modern web technologies and optimized for a seamless user experience.

## 🚀 Live Demo
You can view the live portfolio here: [aaravkhanal-portfolio.vercel.app](https://aaravkhanal-portfolio.vercel.app)

## 🛠️ Tech Stack

**Frontend & UI**
- [Next.js 15](https://nextjs.org/) (App Router, Turbopack)
- [React 19](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS 4](https://tailwindcss.com/)
- Glassmorphism & Premium UI Design

**Animations & 3D**
- [Framer Motion](https://www.framer.com/motion/) (UI Interactions, Staggered Reveals)
- [GSAP](https://gsap.com/) & ScrollTrigger (Scroll-based Timeline Orchestration)
- Cinematic Frame-by-Frame Canvas Scrubbing

**Integrations**
- [EmailJS](https://www.emailjs.com/) (Serverless Contact Form)

## ✨ Key Features

- **Cinematic Experience:** Uses a highly optimized `<canvas>` sequence synced with scroll to deliver an immersive, interactive 3D-like background without the performance overhead of WebGL.
- **Premium Design System:** Linear-inspired glassmorphism, responsive masonry grids, custom typography hierarchies, and curated dark-mode color palettes.
- **Performance Optimized:** Lazy loading, Next.js image optimization, and 60 FPS animations.
- **Interactive UI Components:** 3D parallax hover cards, animated count-up engineering statistics, dynamic skill chips, and functional email integration.

## ⚙️ Local Development

To run this project locally on your machine:

1. **Clone the repository**
   ```bash
   git clone https://github.com/Aaravkhanal/Portfolio-Website.git
   cd Portfolio-Website
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up Environment Variables**
   Create a `.env.local` file for EmailJS credentials if you intend to test the contact form:
   ```env
   NEXT_PUBLIC_EMAILJS_SERVICE_ID=your_service_id
   NEXT_PUBLIC_EMAILJS_TEMPLATE_ID=your_template_id
   NEXT_PUBLIC_EMAILJS_PUBLIC_KEY=your_public_key
   ```

4. **Run the development server**
   ```bash
   npm run dev
   ```

5. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000)
